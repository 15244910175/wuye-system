# wuye-stsyem

## 实现功能

实现业主和管理员两个权限的功能，实现全栈，数据记录在数据库，前后端分离

------

## 项目部署

### 数据库

### 项目初始化
```
npm install
```

### 后端部署和运行
```
cd service
npm install
node app.js
```

### 运行前端项目
```
npm run serve
```
